### 핵심 설계 (이후 Phase의 전제)
- **시뮬 단일 진입점**: `simulate(state, now) → { state, gains }` 순수 함수.
  실시간 틱(200ms)과 오프라인 정산(상한 12시간)이 같은 코드.
- **커맨드 패턴**: 플레이어 조작은 `commands.ts`(startAction/stopAction/sellItem)로만 상태 변경.
- **코어 결정성**: `Date.now()`는 코어 밖에서만. 코어는 (상태, 시각)의 함수.
- **세이브 버전 관리**: `migrateSave()`에 버전별 마이그레이션 체인. 현재 v2.
- **작업 슬롯**: 1슬롯 시작, 총 레벨 30→2슬롯, 90→3슬롯 (`slots.ts`).
  같은 스킬 작업은 교체, 슬롯 가득 시 가장 오래된 작업 교체.

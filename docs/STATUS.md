## 중단된 작업: Phase 2 플레이테스트

확인하려던 것 (아직 못 봄):
1. 장비 없이 슬라임을 잡다가 멈추는가 → "검부터 만들자" 동기가 생기는지
2. 늑대쯤에서 음식 없이는 위험한가 → 위험도 배지(안전/주의/위험)가 체감과 맞는지
3. **던전 패배 → 장비·음식 재정비 → 클리어** 흐름이 재미있는지 (Phase 2 DoD 핵심)

테스트 도우미: `packages/core/scripts/test-save.mjs`
```
cd packages/core
node scripts/test-save.mjs wolf     # 늑대 단계 (철 검+가죽 갑옷+음식 50)
node scripts/test-save.mjs goblin   # 던전 도전 직전 스펙
node scripts/test-save.mjs orc      # 미스릴 단계
```
출력 JSON을 게임의 설정 → 세이브 가져오기에 붙여넣기. (dist 기준이라 데이터 수정 후엔 `npm run build` 먼저)

## 다른 PC에서 환경 재구축

1. 저장소 가져오기 (원격이 없다면 먼저 GitHub 등에 push 필요 — 아래 참고)
2. `npm install` (Node 20+)
3. `npm run dev` → http://localhost:5173
4. 게임 세이브는 localStorage라 PC 간 이동 안 됨 → 설정 → 내보내기/가져오기 또는 test-save 사용
5. 검증: `npm test` (32개), `npm run typecheck`, `npm run balance -w @idle-rpg/core`

## 커밋 대기 중인 변경 (Phase 2 전체)

커밋 메시지:
```
feat: Phase 2 — 전투와 던전 (결정적 기대값 전투 + 시드 난수 전리품)

- 전투 스킬(공격/체력) 추가, 몬스터 6종 사냥터, 위험도 배지(안전/주의/위험)
- 장비 시스템(무기/방어구): 방어구는 전투 드랍(가죽)으로 제작 → 전투-제작 되먹임
- 사냥 = 슬롯 작업: 음식 자동 섭취, HP 부족 시 죽기 전 중단(소프트 페널티)
- 결정성: 전투는 난수 없는 기대값, 전리품은 처치 순번 시드
  → 틱 누적과 오프라인 일괄 정산 완전 동일 (테스트 보장)
- 던전 고블린 소굴: 단발 도전 + 쿨다운 10분, 보상 미스릴 강화석이 티어 관문
- UI: 캐릭터/사냥터/던전/도감 패널, 상단 바 HP·총 레벨 진행도 표시
- 세이브 v3 마이그레이션, 슬롯 해금 기준 40/100, 테스트 17 → 32개
- 개발용 test-save 생성기, phase-5 문서에 전투 뷰 항목 추가
```
+ `.claude/`는 개인 설정이므로 `.gitignore`에 추가 권장: `echo .claude/ >> .gitignore`

## 합의된 작업 방식 (다른 환경의 Claude를 위해)

- 커밋은 사용자가 직접. Claude는 **한국어 커밋 메시지 제안만**
